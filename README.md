#Aula Git, Code Education

##Passos realizados para criação



* Criado um novo repositório no Github

* Criado uma nova pasta do projeto localmente

* Usando o Git Bash, acessei o diretório do meu projeto local e iniciei um repositório com o comando **git init**

* Criei "README.md" para atender a fase 1 do projeto e fiz commit utilizando o comando **git add README.md -m 'Criando o Arquivo README'**
* Sincronizei o meu projeto local com o diretório remoto utilizando **git remote add origin https://github.com/leandroapcorrea/aula-git.git**
